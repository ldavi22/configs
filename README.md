# Configs

My personal configuration files for Neovim and Tmux.

## Contents

- `nvim/` — Neovim configuration
- `tmux/` — Tmux configuration

## Setup

Clone the repo and symlink the configs to the appropriate locations:

```bash
git clone https://github.com/ldavi22/configs.git
ln -s ~/configs/nvim ~/.config/nvim
ln -s ~/configs/tmux ~/.config/tmux
```
